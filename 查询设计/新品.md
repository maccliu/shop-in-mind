# 新品

列出最近的10个新品。

条件：纽澳用户，从新西兰仓发货

```sql
SELECT
    `bc_stu`.`stu_id`,
    `bc_stu`.`title`,
    `bc_stu`.`weight`,
    `bc_stu`.`image`,
    `bc_stu`.`thumb`,
    `bc_stu`.`detail_id`,

    `pricelist`.`price_type`,
    `pricelist`.`user_rank`,
    `pricelist`.`warehouse_id`,
    `pricelist`.`currency`,
    `pricelist`.`price`,
    `pricelist`.`start_at`,
    `pricelist`.`end_at`,
    `pricelist`.`post_free`,
    `pricelist`.`updated_at`
FROM
    `bc_stu`
INNER JOIN
    (
        SELECT
            *
        FROM
            `bc_stu_price_view`
        WHERE
            `warehouse_id` IN (2, 0)
            AND `user_rank` IN (6, 0)
            AND ((`start_at` IS NULL) OR (`start_at` <= CURRENT_TIME))
            AND ((`end_at` IS NULL) OR (`end_at` >= CURRENT_TIME))
        GROUP BY
            `stu_id`
    ) AS `pricelist`
        ON
            `bc_stu`.`stu_id` = `pricelist`.`stu_id`
WHERE
    `bc_stu`.`deleted` = 0
    AND `bc_stu`.`on_sale` = 1
ORDER BY
    `bc_stu`.`stu_id` DESC
LIMIT
    10
```
